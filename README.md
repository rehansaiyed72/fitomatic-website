# Fitomatic Fitness Club Website

A modern fitness club website built using HTML, CSS, and JavaScript.

## Features

* Modern dark-themed UI
* Responsive design
* Fitness Assessment (BMI Calculator)
* Membership Plans
* Testimonial Slider
* Contact & Instagram Integration
* Mobile-friendly layout
